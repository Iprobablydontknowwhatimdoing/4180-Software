Get 4180 Software running on your machine in just a few minutes.

## Prerequisites

- Node.js 18+
- npm or yarn
- 5 minutes of free time ⏱️

## 1. Installation (1 minute)

```bash
# Clone the repository
git clone https://github.com/Iprobablydontknowwhatimdoing/4180-software.git
cd 4180-software

# Install dependencies
npm install
```

## 2. Configuration (1 minute)

```bash
# Copy example environment
cp .env.example .env.local

# Edit with your settings (optional for quick start)
# nano .env.local
```

## 3. Start Development Server (1 minute)

```bash
npm start
```

## 4. Explore Features (2 minutes)

```bash
# Create your first module
npm run create-module my-first-module

# See it in action
npm run dev
```

Visit `http://localhost:3000` and navigate to your new module!

## Common Commands

```bash
# Start development server with watch mode
npm run dev

# Build for production
npm run build

# Run tests
npm run test

# Format code
npm run format

# Lint for issues
npm run lint
```
